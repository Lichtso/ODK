= Class: GUIButton
GUIButton

=== Parent Class
link:GUIFramedView.html[GUIFramedView]

=== Constructor: parent
_Type:_ GUIView +
The parent view

=== Property: sizeAlignment
_Type:_ Number +
0 for none, 1 for width, 2 for height, 3 for both

=== Property: paddingX
_Type:_ Integer +
Horizontal padding

=== Property: paddingY
_Type:_ Integer +
Vertical padding

=== Property: enabled
_Type:_ Boolean +
Wether the button is enabled for user input or not

=== Property: state
_Type:_ String +
0 fro released, 1 for highlighted, 2 for pressed

=== Property: type
_Type:_ String +
0 for normal, 1 for delete button (red), 2 for add button (green), 3 for edit button (blue), 4 for lockable button (checkbox or tabs)

=== Property: onclick
_Type:_ Function +
Anonymous function that is called when this element is clicked
