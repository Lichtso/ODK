= Class: GUITabs
GUITabs

=== Parent Class
link:GUIView.html[GUIView]

=== Constructor: parent
_Type:_ GUIView +
The parent view

=== Property: sizeAlignment
_Type:_ Number +
0 for none, 1 for width, 2 for height, 3 for both

=== Property: orientation
_Type:_ Number +
Bitfield: 1 for left, 2 for right, 3 for vertical, 4 for bottom, 8 for top, 12 for horizontal

=== Property: selected
_Type:_ Integer +
Index of selected child view

=== Property: deactivatable
_Type:_ Boolean +
Determines if it is possible to not have any tab selected

=== Property: onchange
_Type:_ Function +
Anonymous function that is called when the active tab is changed
