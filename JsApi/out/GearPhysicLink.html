Class: GearPhysicLink

GearPhysicLink

Represents two gears. Doesn't fix the objects in place, you are free to place them wherever you want.

new GearPhysicLink(objectA, objectB, axisA, axisB, ratio)

GearPhysicLink
Parameters:
Name Type Description
objectA RigidObject The first object
objectB RigidObject The second object
axisA Vector3 The axis the first object rotates around
axisB Vector3 The axis the second object rotates around
ratio Number The ratio of the gears radii (objectA to objectB). 1 means that they rotate equally.
