HingePhysicLink

Represents a hinge.

new HingePhysicLink(objectA, objectB, frameA, frameB)

HingePhysicLink
Parameters:
Name Type Description
objectA RigidObject The first object
objectB RigidObject The second object
frameA Matrix4 A matrix as description of the axis of the hinge relative to objectA
frameB Matrix4 A matrix as description of the axis of the hinge relative to objectB
Properties:
Name Type Description
hingeAngle Number (READONLY) The current angle between both objects (measured relative to the initial position)
angularLimitMin Number The minimal hingeAngle as a hard limit
angularLimitMax Number The maximal hingeAngle as a hard limit
angularMotorEnabled Boolean Activates the motor around the axis if set to true
angularMotorVelocity Number The velocity of the motor around the axis
angularMotorForce Number The maximal force of the motor around the axis
